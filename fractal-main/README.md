# FRACTAL (Sentinel Protocol)
**Recursive Hierarchical Grammar Induction Engine**

Team Shashwat | SIES GST - B.Tech AIML '27

FRACTAL eliminates the "Observability Tax" by using client-side grammar induction to compress log structures by up to 100x. By processing meaning instead of just bytes before data ever touches the network, FRACTAL trades zero-cost client CPU cycles for expensive cloud bandwidth and carbon-intensive storage.

## The Architecture
* **Rust (WASM) Core:** Executes Fixed-Depth Prefix Tree routing and 2D Forest Straight-Line Programs (FSLP) within the V8 engine, strictly bound by a 10MB stateful streaming limit with Entropy Bypass. 
* **Next.js Edge Command Deck:** A 100vh "Cyberpunk Mission Control" visualizing data vaporization, real-time schema deduplication, and projective data retrieval. 

## GreenOps Impact
By offloading compute to the edge, FRACTAL prevents approximately 5.0kg of CO2 per Terabyte of data transmission and operates on an Edge Arbitrage SaaS rate of $1.50/GB.
